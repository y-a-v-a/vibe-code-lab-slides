💬 Prompting Strategies

🗣️ Natural Language

Explain to AI like you would to a friend or neighbor 👥

💡 Useful Phrases

  • 🎨 "Use an HTML canvas element"
  • 🎲 "Add randomness"
  • 🔢 "Give me X options to choose from"
  • 🌐 "Keep in mind to use standard web APIs only"

🎛️ Meta-Features

Ask AI to add controls: 🎮

  • 🔄 Press 'R' to refresh/regenerate
  • 🎚️ Sliders for parameters
  • 🎨 Color pickers
  • 🎲 "Entropy" or randomness controls
  • ⌨️ Text inputs for live changes

📚 Documentation Approach

🛤️ Two Paths:

  1. 🔨 Build first → Ask for README.md
  2. 📝 Write README.md → Ask AI to build it

🎨 Creative Strategies

✨ The Art of Vibe Coding

🏺 The Archaeological Approach

Starting with found code and excavating/reimagining them 🔍

  • "What if this old Processing sketch was 3D?" 📐
  • "What if this responded to audio?" 🎵

📦 Constraint Boxing

Deliberately imposing arbitrary limitations 🚧

  • 🎨 "Only use three colors"
  • 📏 "No curves allowed"
  • 🐦 "Must fit in a tweet"
  • 📊 "Only mathematical functions from 1970"

⚡ The constraint becomes the creative engine

🔀 The Remix Strategy

Taking two unrelated concepts and forcing them together 🤝

  • 🔬 "Conway's Game of Life meets Arabic calligraphy"
  • 📈 "Stock market data as Medieval manuscripts"

💭 Emotional Debugging

Using coding as mood regulation 🧘

  • 💥 Chaotic particle systems when frustrated
  • 🌊 Gentle wave functions when contemplative
  • 📐 Rigid geometric patterns when seeking control

🌊 Chaos Surfing

Challenge the AI to "go full entropy" 🎲

  • 🔥 Crank creativity dial to maximum
  • 🤗 Embrace unexpected outputs
  • ⛏️ Mine randomness for creative seeds

💭 Like accessing the AI's dream state

🤝 Collaborative Consciousness

Treating AI as genuine creative partner 🤖

The conversation itself becomes part of the artistic process 💬

✨ Generating ideas neither human nor AI would reach alone

🔄 Responsive Evolution

Each iteration responds to what the previous step revealed 👂

Not following predetermined plans but listening to what the code tells you 🎵

🐛 Bugs become creative prompts

🧠 Mental Composting

Ideas ferment in your subconscious 🌱

Like aging wine or making sourdough 🍷🍞

⏰ Time and mental chemistry do the work
